## Resumen Ejecutivo

Trama Educativa es un portal de noticias de última generación, diseñado específicamente para la comunidad educativa de la región. Desarrollado con las mismas tecnologías que utilizan los principales medios digitales del mundo, ofrece una experiencia de usuario superior, accesible desde cualquier dispositivo.

**VERUMax** pone a disposición de la Cooperativa Minga una solución tecnológica integral que posiciona a Trama Educativa a la altura de portales como Infobae, La Nación o El País de España.

---

## ¿Por qué Trama Educativa es Diferente?

### Ventajas Competitivas frente a otros Portales de Noticias

| Característica         | Portales Tradicionales    | Trama Educativa                                                  |
| ---------------------- | ------------------------- | ---------------------------------------------------------------- |
| Búsqueda               | Solo por palabras exactas | **Búsqueda Semántica con IA** - Encuentra por significado        |
| Artículos relacionados | Por categoría únicamente  | **IA que analiza contenido** y sugiere notas realmente similares |
| Accesibilidad          | Básica o inexistente      | **Modo oscuro, TTS, ajuste de texto**                            |
| Interacción            | Comentarios tradicionales | **Reacciones rápidas** sin necesidad de registro                 |
| Velocidad              | Carga pesada              | **Optimizado**: lazy loading, caché inteligente                  |
| Diseño                 | Grillas tradicionales     | **Bento Grid**: diseño modular moderno                           |
| Panel Admin            | Carga manual de datos     | **Asistente IA** para meta tags, descripciones y SEO             |

**La búsqueda semántica es una tecnología que solo implementan los grandes medios internacionales.** Permite que los lectores encuentren artículos buscando por conceptos, no solo por palabras exactas. Por ejemplo, buscar "problemas salariales docentes" encontrará artículos sobre "paritarias", "aumento a maestros" o "reclamos gremiales" aunque no contengan las palabras exactas de la búsqueda.

## Funcionalidades Destacadas

### 1. Búsqueda Inteligente con IA (Exclusivo)

- **Búsqueda por texto**: Búsqueda tradicional por palabras clave
- **Búsqueda semántica**: Utiliza la Inteligencia Artificial de VERUMax para encontrar artículos por *significado*, no solo palabras exactas
- Muestra porcentaje de coincidencia para resultados inteligentes

> *VERUMax integra los mejores modelos de IA del mercado (OpenAI, Anthropic Claude, Google Gemini) para ofrecer resultados de máxima calidad.*

### 2. Artículos Relacionados Inteligentes

- Sistema de IA que sugiere artículos similares basándose en el contenido real, no solo en la categoría
- Mejora el tiempo de permanencia y engagement del lector
- **Tecnología utilizada por medios como The New York Times y BBC**

### 3. Diseño Visual Moderno

| Elemento                 | Descripción                                                                  |
| ------------------------ | ---------------------------------------------------------------------------- |
| **Bento Grid**           | Diseño modular tipo mosaico, tendencia en los principales medios digitales   |
| **Cards Interactivas**   | Tarjetas con efectos hover, sombras dinámicas y transiciones suaves          |
| **Skeleton Loading**     | Animación de carga elegante mientras se cargan las imágenes                  |
| **Tipografía Adaptable** | Ajuste de tamaño de texto (A- / A / A+ / A++) para lectura cómoda            |
| **Animaciones**          | Microinteracciones y efectos visuales (emojis flotantes en reacciones, transiciones) |
| **Modo Oscuro/Claro**    | Cambio de tema con un clic, recuerda la preferencia del usuario              |

### 4. Accesibilidad Universal

| Función                | Descripción                                                         |
| ---------------------- | ------------------------------------------------------------------- |
| Modo Oscuro            | Tema claro/oscuro con memoria de preferencia del usuario            |
| TTS (Text-to-Speech)   | Escuchar artículos en voz alta, con voz argentina/latina automática |
| TTS Premium (Opcional) | Voces naturales de alta calidad con tecnología neuronal             |
| Ajuste de Texto        | Cuatro niveles de tamaño de fuente para lectura cómoda              |
| Barra de Progreso      | Indicador visual de avance en la lectura del artículo               |
| Lista de Lectura       | Guardar artículos para leer después (sin necesidad de registro)     |

### 5. Engagement e Interacción

- **Reacciones**: Los lectores pueden reaccionar a las notas (Informativo, Me gusta, Importante, Para pensar) sin registrarse
- **Efectos visuales**: Confetti animado al reaccionar, feedback visual inmediato
- **Compartir**: Botones optimizados para X (Twitter), Facebook, WhatsApp, LinkedIn y copiar link
- **Ticker de Noticias**: Cinta de noticias destacadas con actualización en tiempo real

### 6. Performance y Optimización

| Optimización         | Beneficio                                                       |
| -------------------- | --------------------------------------------------------------- |
| Lazy Loading         | Las imágenes cargan bajo demanda, página inicial ultra rápida   |
| Page Loader          | Transición visual profesional con logo animado durante la carga |
| Caché Inteligente    | Respuestas instantáneas en búsquedas repetidas                  |
| Imágenes Responsivas | Adaptadas automáticamente a cada dispositivo                    |
| Compresión de Assets | CSS y JavaScript optimizados para carga rápida                  |

### 7. SEO y Visibilidad en Buscadores

- **Meta tags optimizados**: Cada página preparada para Google
- **Open Graph**: Vista previa enriquecida en Facebook y redes sociales
- **Twitter Cards**: Vista previa optimizada para X
- **Feed RSS**: Sindicación de contenido para lectores de noticias
- **Sitemap automático**: Indexación eficiente por buscadores
- **URLs amigables**: Estructura limpia y descriptiva

### 8. Panel de Administración con Inteligencia Artificial

El equipo editorial cuenta con un panel de gestión potenciado con IA:

**Gestión de Contenido:**

- Crear, editar y programar publicación de artículos
- Editor de texto completo con formato enriquecido
- Gestionar categorías, etiquetas y autores
- Subir y gestionar galería de imágenes
- Ver estadísticas de visualizaciones en tiempo real

**Asistente de IA Integrado:**

- **Generación automática de meta descripción**: La IA analiza el contenido y genera descripciones optimizadas para SEO
- **Sugerencia de títulos SEO**: Alternativas de títulos optimizados para buscadores
- **Extractos automáticos**: Genera resúmenes del artículo con un clic
- **Sugerencia de etiquetas**: La IA sugiere tags relevantes basados en el contenido
- **Generación de embeddings**: Activa la búsqueda semántica para cada artículo
- **Optimización de imágenes**: Sugerencias de texto alternativo para accesibilidad

> *El asistente de IA reduce significativamente el tiempo de carga de artículos y mejora el posicionamiento en buscadores de forma automática.*

## ADDENDUM - 21 de Enero de 2026

### Nueva Funcionalidad: Resúmenes IA por Artículo

Se implementó una nueva característica de Inteligencia Artificial que genera automáticamente resúmenes en formato de puntos clave para cada artículo publicado.

#### ¿Qué es?

Un cuadro destacado que aparece al inicio de cada artículo con 3-4 puntos que resumen lo esencial de la nota. Generado automáticamente por IA al momento de publicar.

#### Ejemplo visual:

```
┌─────────────────────────────────────────────────┐
│ ⚡ Resumen IA                           [Beta]  │
│                                                 │
│ • Gremios rechazan oferta salarial del 15%     │
│ • Propuesta en tres tramos no cubre inflación  │
│ • Posible paro de 48 horas la próxima semana   │
│ • Afectaría a más de 15.000 escuelas           │
└─────────────────────────────────────────────────┘
```

#### Beneficios:

| Beneficio | Descripción |
|-----------|-------------|
| **Mejor UX** | El lector sabe rápidamente de qué trata la nota |
| **Mayor engagement** | Decisión informada de seguir leyendo |
| **Accesibilidad** | Facilita la lectura rápida (scanning) |
| **Diferenciación** | Tecnología que usan medios como BBC, The Guardian |
| **SEO** | Contenido estructurado que mejora posicionamiento |

#### Costo adicional:

| Concepto | Costo |
|----------|-------|
| VERUMax IA - Resúmenes | ~$0.02 USD por artículo |
| Estimado mensual (50 artículos) | ~$1 USD |

> *El costo de los resúmenes IA se integra al módulo de IA existente y se suma al consumo general de la API de VERUMax IA.*
